HTML Elements

Article, Section, Aside

The <section> element (outlined in blue) contains a generic, standalone section of a document.

Article

The <article> element (outlined in red) is a self-contained composition, which is intended to be independently distributable or reusable. Examples include:

  • a forum post,
  • a magazine or newspaper article,
  • a blog entry,
  • a product card,
  • a user-submitted comment,
  • an interactive widget or gadget,
  • any other independent item of content.

Aside

The <aside> element represents a portion of a document whose content is indirectly related to the main content.

Figure

The figure element represent self-contained content, with an optional caption.

HTTP Status Codes

The <footer> element represents a footer for a section of content and often contains information about the author, copyright, or links to related documents.
